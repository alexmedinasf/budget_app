# 📖 CashTrack <a name="about-project"></a>

Budget App is a personal finances App, that allow you to track your outgoings.

# Check the live demo [here](https://www.loom.com/share/9b139f84e922416a820fef9eff242d89)

### Key Features <a name="key-features"></a>

- **Create your user account and track your finances**
- **Create customs categories to separate your outgoings**
- **Add every transaction you made**
